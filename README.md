# Word Dictionary Application

A command-line dictionary app built in Python that fetches definitions, examples, and synonyms.

## View the code
The full code is in `word_dictionary.py` above.

## About
Built a fully functional command-line dictionary application in Python that allows users
to search for word definitions, synonyms, and usage examples using the Free Dictionary API.

## Features
- Fetch definitions for any English word
- Displays part of speech for each definition
- Usage examples where available
- Synonyms displayed for each word (up to 5)
- Handles misspellings and non-existent entries with clear error messaging
- Robust error handling for network timeouts and connection issues
- Case-insensitive search for improved user experience

## Tech
Python · REST API · Free Dictionary API · JSON parsing · OOP · Error handling
